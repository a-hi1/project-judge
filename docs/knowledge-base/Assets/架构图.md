## 数据流架构

```
用户输入
  │
  ├─ GitHub URL ──→ fetchGitHubRepo() ──→ repoInfo
  │                                         │
  ├─ 详细描述 ────→ (无额外处理)              │
  │                                         │
  └─ 一句话想法 ──→ (无额外处理)              │
  │                                         │
  ▼                                         ▼
getSystemPrompt(type, repoInfo) ────→ 系统 Prompt
  │
  ▼
DeepSeek API ────→ AI 原始输出 (JSON)
  │
  ▼
extractJson() ────→ 解析后的对象
  │
  ▼
calculateBenchmark() ────→ 覆盖 benchmark 字段
  │
  ▼
返回给前端 { success: true, data: result }
  │
  ▼
transformResponse() ────→ EvaluationResult
  │
  ├─ clampScore() × 6
  ├─ normalizeVerdict()
  ├─ normalizeEstimatedDevTime()
  ├─ parseSimilarProjects()
  ├─ parseBenchmark()
  ├─ parseRoadmap()
  ├─ parseInterviewQuestions()
  ├─ parseResumeStar()
  ├─ parseDevelopmentBreakdown()
  ├─ parseGitHubAnalysis()
  └─ fillDefaultValues()
  │
  ▼
页面渲染
```
